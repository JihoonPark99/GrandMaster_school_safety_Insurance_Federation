### 배경
저출산으로 인해 학생인구가 빠른 속도로 줄어듦에도, 그들의 안전사고수는 급격하게 상승한다. 

정부의 노력으로 학생 안전사고를 위한 예산 증원에도 불구하고, 안전사고 수는 매해 지속적으로 증가하는 추세이다. 

학교안전공제회에서 주관하는 학교안전사고데이터 경진대회를 통해서, 근본적인 학생안전사고의 원인을 도출하고 해결전략을 수립한다.
